# HTML Template
A super simple template to build super simple sites on. I use this for splash / lead pages, and host on Netlify for easy form handling.

## Features
- File architecture
- Common meta tags
  - Seo
  - Social cards
  - Page behavior
- SCSS Setup
  - Useful partials
  - Typescale
  - Webfonts
  - Common variables
  - Css reset
- Basic markup
